# Google Satellite Downloader

## Usage

```bash
python app.py [config.json]
```

## configure file options

```json
{
    "level": 16,
    "lon_km": 102.861464,
    "lat_km": 24.870115,
    "lon": 118.620512,
    "lat": 27.874906,
    "lat_": 27.873906,
    "radius": 1000,
    "key": "YOU-GOOGLE-STATIC_MAP_KEY",
    "out": "out.jpeg",
    "map_type": "satellite",
    "format": "jpg-baseline",
    "scale": 1,
    "resume": false,
    "debug": true
}
```

unit of radius is meter
